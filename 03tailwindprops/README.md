# 03 Tailwind Props

## Description

A React project demonstrating the integration of Tailwind CSS with React components and props. The application showcases a reusable `Card` component that accepts props (`userName` and `message`) to display personalized message cards. This project illustrates how to use Tailwind CSS utility classes for styling and how to pass and use props in React components.

## Features

- Tailwind CSS integration
- Reusable Card component with props
- Default prop values
- Modern UI styling with Tailwind utilities
