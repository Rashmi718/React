# 05 Password Generator

## Description

A comprehensive password generator application built with React and Tailwind CSS. Users can generate secure passwords with customizable options including password length (6-50 characters), numbers, special characters, and regular characters. The app features a copy-to-clipboard functionality with visual feedback, and automatically regenerates passwords when settings change. This project demonstrates advanced React hooks usage including `useState`, `useRef`, `useCallback`, and `useEffect`.

## Features

- Customizable password length (6-50 characters)
- Toggle options for numbers, special characters, and letters
- Copy to clipboard functionality
- Visual feedback when password is copied
- Automatic password regeneration on setting changes
- Modern UI with Tailwind CSS
