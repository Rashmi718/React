# 04 Background Changer

## Description

An interactive background color changer application built with React and Tailwind CSS. Users can change the background color of the entire screen by clicking on predefined color buttons (black, red, pink, blue, olive, lavender, yellow, brown) or by using a custom color input prompt. The app demonstrates dynamic styling, state management, and user interaction in React.

## Features

- Dynamic background color changes
- Multiple predefined color options
- Custom color input via prompt
- Tailwind CSS for styling
- React state management for color updates
