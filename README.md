# React Projects Collection

This repository contains a collection of React projects that demonstrate various concepts and features of React development, from basic setups to more advanced applications.

## Overview

This folder contains 8 React projects, each focusing on different aspects of React development:

1. **01basicreact** - Basic React project using Create React App
2. **01vitereact** - Basic React project using Vite
3. **02counter** - Counter application with state management
4. **03tailwindprops** - Tailwind CSS integration with props
5. **04bgchanger** - Interactive background color changer
6. **05password_generator** - Password generator with customizable options
7. **currencyConverter** - Currency converter with API integration
8. **customReact** - Custom React renderer implementation

## Technologies Used

- **React** - JavaScript library for building user interfaces
- **Vite** - Fast build tool and development server
- **Create React App** - React project setup tool
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript/JSX** - Programming language and syntax extension

## Learning Path

These projects follow a progressive learning path:

1. **Getting Started** (01basicreact, 01vitereact)
   - Basic React setup
   - Component creation
   - JSX syntax

2. **State Management** (02counter)
   - useState hook
   - Event handling
   - Conditional rendering

3. **Styling & Props** (03tailwindprops, 04bgchanger)
   - Tailwind CSS integration
   - Component props
   - Dynamic styling

4. **Advanced Hooks** (05password_generator)
   - Multiple React hooks (useState, useRef, useCallback, useEffect)
   - Complex state management
   - User interaction patterns

5. **API Integration** (currencyConverter)
   - Custom hooks
   - API calls with fetch
   - useEffect for side effects

6. **Understanding React Internals** (customReact)
   - Custom renderer implementation
   - Understanding React's core concepts

## Getting Started

Each project has its own README file with specific instructions. To run any project:

1. Navigate to the project directory
2. Install dependencies: `npm install`
3. Start the development server: `npm start` or `npm run dev`
4. Open your browser to the provided URL (usually `http://localhost:3000` or `http://localhost:5173`)
