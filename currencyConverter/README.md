# Currency Converter

## Description

A currency converter application built with React, Vite, and Tailwind CSS. This project demonstrates API integration, custom hooks, and component-based architecture. The app uses the ExchangeRate API to fetch real-time currency conversion rates. It includes a custom hook (`useCurrencyInfo`) for fetching currency data and an `InputBox` component for user input. The project is currently in development and showcases React hooks, API calls, and component reusability.

## Features

- Real-time currency conversion using ExchangeRate API
- Custom React hook for currency data fetching
- Reusable InputBox component
- Tailwind CSS for styling
- API integration with useEffect
